GIT Flow

GITclone - для того, чтобы "склонить" репоризиторий

GIT clone - для того чтобы получить изменения с репозитория

GIT checkout-b branchname-создать новую ветку и переключится на неё

GIT add - добавить изменения которые сделали локально

GIT commit-m"commit text" - подписать их

GIT push origin branchname - запушить на git сервер
